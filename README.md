# Astro SaaS starter

Create a new Astro project using this starter with this command:

```bash
npm create astro@latest my-astro-project -- --template guigui64/astro-saas-starter
```

## Features

- Login with Github or Google
- Dark mode that matches the OS
- ...

## Tech

- Astro
- Astro DB
- Lucia
- Tailwind
- React
- shadcn/ui
- ...
